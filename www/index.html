Allowlist

If at least one allow rule applies to a host, ONLY listed IPs/ranges are allowed through that host. Leave empty (or scope to other hosts) to disable allowlisting elsewhere.

IP or CIDRHosts (glob, comma-sep; empty = all)

Blocklist

IP or CIDRHosts (glob, comma-sep; empty = all)

User-Agent blocklist

Regex (case-insensitive prefix: (?i)). Matched against the request's User-Agent header.

PatternHosts (glob, comma-sep; empty = all)

WAF rules

Go regex, matched against request URI, full URL, Cookie, and Referer.

OnNamePatternHosts

Per-IP rate limit

Limits each source IP globally across all hosts the plugin is assigned to.

General

Enable when Guardian is behind Zoraxy (the normal case). Disable only if you don't trust the upstream chain.

Host pattern syntax

example.com — exact match.
*.example.com — single subdomain level (does not match a.b.example.com).
**.example.com — any subdomain depth.
* or empty — any host.
